
# Lotto Number Generator

## Overview

This is a simple web application that generates lottery numbers. Users can click a button to generate a new set of 6 unique numbers between 1 and 45. The 10 most recent sets of numbers are stored and displayed.

## Project Structure

*   `index.html`: The main HTML file.
*   `style.css`: The stylesheet for the application.
*   `main.js`: The JavaScript code for the application logic.
*   `blueprint.md`: This file, outlining the project.
